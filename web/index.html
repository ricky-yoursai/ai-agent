Get Started

Install the Extension

Aurora runs as a Chrome extension — it works on every page you visit.

1

Open Extensions Page

Type chrome://extensions in your address bar

2

Enable Developer Mode

Toggle Developer mode in the top-right corner

3

Load Unpacked

Click Load unpacked and select the aurora/extension folder

✨ After installing, click the Aurora icon (✦) on any page to open the chat panel and start giving commands!
